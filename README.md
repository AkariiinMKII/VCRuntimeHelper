# VCRuntimeHelper

Inspired by https://msvc.win/

## How to use

Run following command in PowerShell:

```powershell
$VCRuntimeHelper = "https://raw.githubusercontent.com/AkariiinMKII/VCRuntimeHelper/refs/heads/main/VCRuntimeHelper.ps1"; irm $VCRuntimeHelper | iex
```
